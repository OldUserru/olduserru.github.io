📋 Инструкция по настройке DNS в Windows

  1. Нажмите Win + R, введите ncpa.cpl и нажмите Enter
  2. Найдите ваше активное подключение, кликните правой кнопкой → Свойства
  3. Выберите IP версии 4 (TCP/IPv4) → Свойства
  4. Включите Использовать следующие адреса DNS-серверов
  5. Введите: --- (предпочитаемый) и --- (альтернативный)
  6. Нажмите OK → перезагрузите браузер
